## Preview the landing page

Run a simple local server and open http://localhost:8000 in your browser:

```bash
bash preview.sh
```

The repo now includes a placeholder logo at [assets/logo.svg](assets/logo.svg#L1).

## Deployment

This repository is configured to deploy the site to GitHub Pages automatically on pushes to `main` using GitHub Actions. The workflow file is at [.github/workflows/gh-pages.yml](.github/workflows/gh-pages.yml#L1).

Expected Pages URL after first successful deployment:

https://nilnimi1970-crypto.github.io/NIlesh-/
